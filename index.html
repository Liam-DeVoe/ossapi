ossapi#

ossapi is an api wrapper for osu api v2 and api v1.

ossapi is developed and maintained by tybug.

Installation#

ossapi can be installed from pip:

$ pip install ossapi

Links#

Github: https://github.com/circleguard/ossapi
Documentation: https://circleguard.github.io/ossapi/
Discord: https://discord.gg/VNnkTjm
